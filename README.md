# Web Image Converter (Batch Processing)

Это одностраничное клиентское веб-приложение (SPA) для пакетной конвертации изображений в браузере. Приложение работает полностью на клиенте, не отправляя файлы на сервер, что делает его безопасным и конфиденциальным.


## Основные возможности
* **Пакетная обработка:** Возможность загружать множество изображений за раз, просматривать их статус в очереди и конвертировать их последовательно.
* **Экспорт в ZIP:** Скачивание всех успешно конвертированных изображений одним ZIP-архивом.
* **Форматы:** Конвертация в JPG, JPEG, PNG и WebP.
* **Качество:** Настройка степени сжатия для JPEG и WebP (индивидуально для всех файлов пакета).
* **Прозрачность:** Автоматическая замена прозрачного фона на белый при конвертации в JPEG.
* **Индивидуальная статистика:** Наглядное сравнение размера "До -> После" и процент сжатия для каждого изображения в списке.
* **Drag & Drop:** Удобная зона загрузки с возможностью добавления новых файлов в существующую очередь.
* **Адаптивность:** Полная поддержка мобильных устройств и десктопов (динамическое расширение макета при работе со списком).

## Как запустить проект локально

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Откройте в браузере адрес:
   **[http://localhost:5173/](http://localhost:5173/)**

## Сборка для публикации (GitHub Pages)

Для сборки оптимизированной версии проекта выполните:
```bash
npm run build
```

Собранные файлы появятся в директории `dist/`. Пути к ресурсам настроены как относительные (`base: './'`), поэтому проект готов к публикации на GitHub Pages.
